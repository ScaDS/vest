# Butterfly images

This example uses a dataset of [butterfly images](https://www.kaggle.com/datasets/phucthaiv02/butterfly-image-classification) published on kaggle by DePie.

Go through the [download notebook](download.ipynb) to download the data.
Optionally, you can re-generate the embeddings using [vision_embeddings_umap.ipynb](vision_embeddings_umap.ipynb), but they are also given in [data.csv](data.csv).

Afterwards, you can navigate to this folder using the terminal and run this command:
```
vest data.csv --image-path ./images
```
